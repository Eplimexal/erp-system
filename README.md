# Student ERP System

The Student ERP System is a web-based platform designed to simplify academic and administrative workflows for students. It provides a centralized portal where students can access important information and manage academics efficiently.

# Features

Student dashboard

Attendance overview

Timetable display

Exam schedule and marks

Fee information and status

Announcements and notifications

# Tech Stack

React + Vite

Tailwind CSS

# How to Run

Clone the repository

git clone https://github.com/Eplimexal/erp-system.git


Open the project folder

cd erp-system


Install dependencies

npm install


Start the development server

npm run dev
